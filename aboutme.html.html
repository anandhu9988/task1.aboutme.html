Weekly Questions & Answers

  1. What is HTML?
    1. HTML stands for HyperText Markup Language.
  2. What is CSS?
    1. CSS is used for styling HTML pages.
  3. What is a table in HTML?
    1. Tables display tabular data in rows and columns.
  4. How to insert images?
    1. Using the <img> tag with src and alt attributes.
  5. What is an ordered list?
    1. OL tag creates numbered lists.
  6. What is an unordered list?
    1. UL tag creates bulleted lists.
  7. What is colspan?
    1. Merges multiple columns into one.
  8. What is rowspan?
    1. Merges multiple rows into one.
  9. How to add comments in HTML?
    1. Using <!-- comment here --> syntax.
  10. What is the <title> tag?
    1. Sets the page title shown in browser tab.
  11. What is a navigation link?
    1. Links users to another page using <a href="page.html">Text</a>.
  12. What is the difference between internal and external CSS?
    1. Internal CSS is inside <style> tag; external CSS is in separate file linked with <link>.
  13. How to set image size?
    1. Using width and height attributes.
  14. What is a blockquote?
    1. Used to display quoted text with proper indentation.
  15. What is the purpose of <meta charset>?
    1. Defines character encoding for the page (UTF-8 recommended).
